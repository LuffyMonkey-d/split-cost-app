# 旅行費用割り勘アプリ

Next.js + TypeScript + TailwindCSS で作成した、旅行時の費用精算（割り勘）アプリです。

## 機能
- 参加者の追加
- 支払い記録の追加
- 割り勘精算（誰が誰にいくら払えば良いか計算）
- セッション内で完結（DB不要）

## セットアップ

### Dockerで起動
```sh
docker compose up --build
```

### ローカルで起動
```sh
npm install
npm run dev
```

## 工夫点
- UI/UXを意識したシンプルなデザイン
- バリデーション（マイナス値禁止、小数点処理）
- コンポーネント分割しやすい設計
